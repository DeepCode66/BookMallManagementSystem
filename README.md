## 功能介绍（完整见运行截图）
管理员：基本功能包括登录、注册和用户信息管理。网站首页拥有主导航栏和全站概览。可通过菜单管理订单、顾客、图书分类及用户信息，高效管理库存及推荐书目。后台支持在线图书信息的操作与更新。
